# YouTube Shorts Blocker

A Tampermonkey userscript that removes YouTube Shorts from your feed, sidebar, and search results, and redirects Shorts URLs back to your subscriptions feed.

## What it does

- Removes the Shorts shelf from the home page and subscription feed
- Removes the Shorts button from the sidebar
- Removes Shorts from search results
- Redirects any `/shorts/` URL to your subscriptions feed
- Cleans up empty containers left behind so the page layout stays correct

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) for your browser
2. Click this link: [Install Script](https://github.com/zenjaymusic/youtube-shorts-blocker/raw/refs/heads/main/youtube-shorts-blocker.user.js)
3. Click **Install** in the Tampermonkey prompt
4. Refresh YouTube

## Updating

Tampermonkey will check for updates automatically. You can also trigger a manual check from the Tampermonkey dashboard.

## Browser support

Works on any Chromium-based browser (Chrome, Opera GX, Edge, Brave, etc.) with Tampermonkey installed.
